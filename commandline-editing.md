## Cursor Movement

This table can be used for moving the cursor around on the commandline.

| Key | Description |
| --- | --- |
| `ctrl-a` | Moves cursor to the beginning of the line. |
| `ctrl-e` | Moves cursor to the end of the line. |
| `ctrl-f` | Moves cursor forward by one character. |
| `ctrl-b` | Moves cursor backward by one character. |
| `alt-f` | Move cursor forward by one word. |
| `alt-b` | Moves cursor backward by one word. |
| `ctrl-l` | Clears the terminal and moves the cursor to the top. |

## Modifying Text

| Key | Description |
| --- | --- |
| `ctrl-d` | Delete character at the location of the cursor. |
| `ctrl-t` | Transpose the character at the cursor with the one preceding it. |
| `alt-t` | Transpose the word at the cursor with the one preceding it. |
| `alt-l` | Convert characters from the location of the cursor to the end of the word to lowercase. |
| `alt-u` | Convert the characters from the location of the cursor to the end of the word to uppercase. |
