Being able to work with text files is one of the most important skills that a Linux admin can learn. Below is a table of common tools that admins use to manage text files on a Linux system. 

| Command | Description |
| --- | --- |
| cat | Dumps content of a file on to the terminal screen. |
| less | Dislays text in a pager, allowing for navigation and easier reading. |
| head | Displays the top few lines of a text file. |
| tail | Displays the bottom few lines of a text file. | | cut | Filters specific characters or columns from a file. | | Sort | Sorts the contents of a file | | wc | Provides a count of the number of lines, words, and characters in a file. |
